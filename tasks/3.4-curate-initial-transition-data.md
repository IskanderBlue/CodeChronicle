## Why

in the metadata source.

  - BCBC 2024

  - QECB 2020
  - QPC 2020

- Ensure the citation and applicability text in each record is specific enough

- All five known transitions exist in `config/transitions.json`.
- Records pass the validation introduced in 3.1.
- Search tests can reference these records directly.

## How to Test

### Unit test: `test_initial_transition_fixture_contains_five_known_records`

Assertions:

```python
records = load_transitions()
assert len(records) >= 5
assert any(r["new_edition"] == "BCBC_2024" for r in records)
assert any(r["new_edition"] == "QCC_Building_2020" for r in records)
assert any(r["new_edition"] == "QECB_2020" for r in records)
assert any(r["new_edition"] == "QPC_2020" for r in records)
assert any(r["new_edition"] == "QSC_2020" for r in records)
```
