# 4.4 — Viewer Mode: e-Laws Content Display

## Status: future

## What

Display e-Laws (OBC) section content in the viewer overlay instead of hiding
the Document viewer panel. Currently the panel is hidden for non-PDF results.

## Options

1. **Inline HTML** — Load the stored `html_content` for the viewed section and
   its siblings/neighbours into the viewer panel. Requires an API endpoint that
   returns surrounding sections for a given node, similar to how PDF viewer
   shows surrounding pages.

2. **e-Laws iframe** — Embed the e-Laws page via `source_url` in an iframe,
   ideally deep-linked to the specific section. Add a companion "Open in new
   tab" link.

Either approach should integrate with the edition navigation already in the
viewer (task 4.2) so switching editions re-fetches the correct content.

## Acceptance

- Viewer opens for OBC results and displays readable section content.
- e-Laws link opens in a separate tab.
- Edition switching works for HTML viewer same as PDF viewer.
